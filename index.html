DISEÑO DE SITIOS WEB

Bienvenido al curso de Diseño de Sitios Web. Aquí encontraras información básica para aprender
de forma dinámica e interactiva los conceptos de HTML5, CSS3, GIT y GITHUB.

OBJETIVO GENERAL

Realizar un Objeto Virtual de Aprendizaje para entorno WEB, con el fin de brindar un proceso complementario,
dinámico e interactivo de enseñanza, a través del curso “Diseños de Sitios WEB - 301122”
por medio de las Tecnologías de la Información y Comunicación.

OBJETIVO ESPECÍFICO 1

Brindar información completa, clara y concisa acerca de las temáticas GIT/GITHUB, HTML5 y CSS3, distribuida en tres capítulos o secciones, para ser visualizadas en snavegadores web de PC y Dispositivos Móviles.

OBJETIVO ESPECÍFICO 2

Presentar al usuario contenidos complementarios con recursos didácticos y multimedios (Videos, Animaciones, Audio, Imágenes), para mejorar la retención cognitiva del usuario y su aprendizaje.

OBJETIVO ESPECÍFICO 3

Crear una interfaz amigable para el usuario, manejando buena teoría de color, tamaño de fuentes, tipografía, botones y transiciones, para brindar una información amena y completa sin perder el interés por aprender.
